CareerPath is a Django-based web application that provides structured career assessment and personalized recommendations for students.

This project focuses on backend architecture design, database modeling, authentication systems, and structured business logic implementation.

---

## Project Overview

The platform allows users to:

- Complete a structured career assessment
- Receive rule-based career recommendations
- Manage personal profiles
- Access subscription-based features
- Authenticate securely using Google OAuth

The system is designed with modular backend architecture to ensure maintainability and scalability.

---

## System Architecture

Backend:
- Django (modular application structure)
- Separation of models, views, and templates
- Organized business logic handling

Database:
- MySQL relational database
- Designed entity relationships for users, assessments, and recommendations

Authentication:
- Google OAuth integration
- Session-based authentication management

Email Services:
- SMTP integration for account-related communication

Frontend:
- Server-side rendering using Django Templates
- Structured and consistent UI layout
